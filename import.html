📖 How to Export From Each Tool
🟠 Zoho Projects
Go to Tasks → Export → Export as CSV. Select all columns. Download the CSV file.
🔵 Jira
Go to Issues → Export → Export XML (for full data) or use Jira's Backup for Cloud JSON export.
🟦 Trello
Open your board → click Show menu → More → Print and Export → Export as JSON.
🟤 Asana
Open your project → click ⋯ → Export/Print → CSV. Download the file.
